# 🔎 Auditoria MFA LATAM – Engenharia Forense Web (Semana 1)

Projeto técnico de auditoria do fluxo de autenticação da LATAM Airlines, com foco em:
- Análise do DOM
- Interceptação de requisições
- Mapeamento de endpoints
- Análise de cookies críticos
- Persistência e fingerprinting
- MFA adaptativo (OTP vs Reconhecimento facial)

Este repositório contém **evidências reais** coletadas através de DevTools (Chrome), seguindo práticas de QA técnico e engenharia reversa de front-end.
